How to Use

The @state private var codes make the numbers entered by the user work in the calculator. The var sum, cik, mul and dev codes perform the calculator’s tasks which are addition, subtraction, multiplication and division. VStack ensures that the texts are reflected vertically. The TextField code allows the user who enters the numbers into the calculator to see the numbers they wrote and to see the place where they can write the number. The Text codes are the codes that allow the displayed result to be reflected.
